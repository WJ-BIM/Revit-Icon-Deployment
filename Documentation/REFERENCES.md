## Icon Naming Convention

See: `DOCUMENTATION/NAMING-CONVENTION.md`

Format: **RVTYY.ico**

- RVT23.ico = Revit 2023
- RVT24.ico = Revit 2024
- RVT25.ico = Revit 2025
- RVT28.ico = Revit 2028 (future)
